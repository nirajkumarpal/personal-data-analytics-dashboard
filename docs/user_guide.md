# User Guide

This guide explains basic usage for an end user.

1. Open the app in the browser (e.g., http://127.0.0.1:5001/).
2. Register a new account via the sign-up page.
3. Add daily logs with study hours, screen time, mood score, and tasks.
4. Visit the Analytics dashboard to see trends, productivity score, and insights.

Tips
- Keep daily entries consistent (same date format) for accurate trends.
